name: iem
description: |
  This skill is used to easily interact with the Iowa Enviornmental Mesonet weather API.
  It is an excellent source of historical hourly weather data.
  It has been wrapped into a simple CLI tool.
---

`iem` is a CLI utility for interacting with the Iowa Environmental Mesonet weather API. It has a simple API, as described by the help.
Use `iem --help` for the up to date signature.

Depending on the environment, it can be invoked with `iem.msh` or `iem`. Try `iem.msh` first.
