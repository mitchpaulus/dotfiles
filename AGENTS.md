These are my dotfiles. Some things of note:

- program configuration is typically in `.config/`


## Python libraries

There are a few one-off Python libraries that haven't been put into their own projects.
They are in the `python` directory.

### `mputils.py`

This is an important catch all of definitions that I find useful in many projects.

### `word_vba.py`

This is a library for creating VBA code that will generate tables in Microsoft Word.

### `diagram.py`

Library that has many primitives for creating SVG diagrams.

### `idf.py`

CLI that helps inspect EnergyPlus `.idf` files.
